brobot.core.irc.clients¶

Contains mid-level building blocks for an IRC client. This is really the bridge between the low-level connections module and the high level client that someone using this library would write.

class brobot.core.irc.clients.Client(servers, event_plugins=None)[source]¶

The base IRC Client, which wraps a ConnectionManager and provides an interface to low level connection functions. It can be extended to make a full IRC client or an IRC bot.
